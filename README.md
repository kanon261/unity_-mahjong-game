## 🎯 プロジェクト概要
「学んで上達できる麻雀アプリ」は、初心者でも楽しみながら麻雀のルールと戦略を学べることを目的にした学習支援型ゲームです。  
Unity を用いて実装しており、ゲームを通して**麻雀の基本ルール・役・点数計算・戦略**を自然に身につけることができます。

---

## 🧩 主な機能

| 機能名 | 内容 |
|--------|------|
| 🀄 **対局モード** | プレイヤー vs CPU の1人プレイ。難易度調整機能あり。 |
| 📚 **学習モード** | 役や点数計算をクイズ形式で学べるモード。 |
| 💬 **チュートリアル** | 初心者向けにルール・操作方法を解説。 |
| 🌈 **スコア履歴管理** | 自分の成績・役成立回数を可視化。 |
| 🎵 **演出・BGM** | Unity Audio を活用した臨場感ある演出。 |
